# AI Study Assistant 🤖

An AI-powered study assistant built using Flask and Google Gemini AI.

## Features

- AI-powered question answering
- Subject selection
- Difficulty level selection
- Chat interface
- Gemini AI integration

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Google Gemini API
